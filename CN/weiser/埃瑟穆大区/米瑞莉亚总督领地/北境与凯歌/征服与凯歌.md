# 泰拉利安半岛
**介绍**：
泰拉利安半岛，这一地理单元位于冰海之北，佩雷迪尔山脉以东。他是一块巨大的，靴形的半岛，向东伸入浩瀚的大洋之中。在南方王国地区，它被习惯性地称呼为“北境”，这无疑是他们最大的谬误，北回归线实则横穿半岛地土地。或许冰海也只是造物主为这片土地设置的面纱，屏障之后，自生命兴盛起，一直便是文明与强国的兴衰之地。这里雨热同期，广袤的平原盛产着长年的丰饶，壮丽的圣岳俯瞰千年的城邦。当地人自视为古老帝国文明的继承者，这些守夜人依然延续着旧日帝国的余晖。
**现状**：
如今，这片半岛的一大部分，已经归属于王国的统治之下，这当然不是一朝一夕间完成的占领，而是来自新秩序锲而不舍的蚕食与替代。彼时，她的前主人，波斯塔特帝国的继统诸国已经在内耗中流干了鲜血，面对南方宿仇的虎视眈眈，西侧山国的甚嚣尘上，古老的同盟早就摇摇欲坠。

历史当然赐予过他们喘息的机会，南方王国的动荡迫使征服的危机延缓。那一张脆弱的合约为半岛上的诸国争取到四十余年的平静。然而，宿命的钟声终究会敲响。随着王国内战的平息，一个东西一心、如日中天的庞大帝国已然成型。它将完成对于波斯塔特这一名号以彻底的终结。
# 各方势力
## 王国势力
### 王冠领地
**介绍**：推翻旧国后，王室给予自己的保留地。自建国以来，王冠领地一直是诸侯中领地最大，势力最广的一方。在结束动乱之后，叛乱领地臣服并入王冠领地，使得王国获得了前所未有的底蕴，正是王冠领地作为引擎的支持，才支撑起王国对于北方征伐的消耗。

### 斯佩比亚领地
**介绍**：斯佩比亚大公领地是南方王国建国时分封的大公领地，一直作为拱御其王都的西北势力而存在，是王国西北的坚盾与利剑。长期以来，斯佩比亚大公领地与其他分封势力同波斯塔特继统诸国发生过数之不尽的边境战争。但在斯佩比亚家底蕴与智谋的支持下，边境的领地在纷扰中最终易帜，原本以大河为界的王国边境一直推到了佩雷迪尔山南麓。

Secovia在母亲退隐后发动建业军与继业军两支大军团西进。以帮助友邦收复失地为理由，击退入侵，抢占爱流城周边地区。米瑞莉亚与来自泰梅里亚伯爵国大贵族联姻。
### 东方领地（名称与具体身份待定）
**介绍**：东方领地一直是海军与经济重心，尽管在内战中失败，但他们却为此次战争提供了前所未有的人力与财政支持。
## 泰拉瑞安半岛势力
### 波斯塔特继统诸国
**介绍**：
波斯塔特继统诸国是大陆上的古老国家，他们几乎与王国同时诞生。文化上又直接传承于前波特塔斯帝国，因此与王国的历史渊源极为密切，这也带来二者间持续不断的冲突。自旧帝国崩亡后，泰拉瑞安各邦奉帝国法统自居，史称**波斯塔特继统诸国**。然其名为同盟，实为自保之约，也从未有过统一。他仍然还是由少数大邦国主导的松散联合体，没有共主的旧国守夜人。
#### 泰梅里亚圣岳国
**介绍**：波斯塔特同盟成员之一。坐拥半岛西部的泰梅里亚平原，该国长期以“圣岳守护者”自居。他的开国者曾经是是圣岳地区的宗教领袖，扩张成为举足轻重的大国，而他的送葬人们最终在大梦一场后也选择回归了这个身份。

在金诺瓦尔求援时，它选择了沉默。在斯佩比亚联姻时，他选择了接受，在阿尔泽兰受难时，它选择了观望。不合时机的骤然下注使得他们丢失了全部的筹码，泰梅里亚的精锐在丰泽一战中几乎葬送殆尽。最终，为了保全自己的家族与爵位，泰梅里亚的贵族们与王国达成了密约——他们出卖了盟友夏尔赞国，为王国军队让开了通往夏尔赞平原的通道。 虽然它最终也向王国投降，但凭借这一系列政治操弄，泰梅里亚的贵族体系得以最大程度地保留，尽管西部平原尽数失去，只剩下圣岳山地，但他们以“合作者”的身份在新秩序中苟延残喘，成为了这场征服中唯一幸存的旧贵族体系。他们在王国的新秩序中，像一件精美但过时的摆设被小心翼翼地供奉着。
```
“先辈利欲熏心，这并非圣岳养时所育。”
```

#### 金诺瓦尔王国
**介绍**：金诺瓦尔是佩雷迪尔山脉南方的贵族领土，即便在帝国崩溃与反叛的浪潮之中，他依然依托故地重整旗鼓，而不似东方那般消亡。他的领土曾到达密洛大河东畔，是波斯塔特继统诸国悬挂在新生王国头顶最锋利的剑。

尽管在数代的兵灾中，金诺瓦尔失去了平原沃土，金诺瓦尔人向山而生，在佩雷迪尔的丛山峻岭中修筑了令王国绝望的防线。其领土更是失之于原，取之于山，开发了佩雷迪尔山中大量的盆地、谷底，甚至染指山脉西部，虽丢失山南故土，但其实力并未减弱。可以说，金诺瓦尔凭借一己之力，几乎将南方王国的兵峰死死挡在山门之外，斯佩比亚的铁骑巨兽在其险峻关隘之前讨不到丝毫好处。甚至达到了称王的威望。

基诺瓦尔的覆灭并非源于正面战场的溃败，而是源自内忧外患下的慢性死亡。流亡贵族没有领地，没有收入，却依然维持着奢靡的排场和巨大的军费开支。这一切都通过“特别战争税”压在了山地贵族的头上。他们坚持把主力部队部署在南部关隘（通往平原的方向），即使那里已经固若金汤。他们甚至策划了几次毫无意义的“反攻”，让山地子弟兵去平原送死。当图拉尔山国异军突起，疯狂劫掠其毫无防备的西部腹地之时，波斯塔特继统诸国的其他成员忌惮其势力，几乎全部冷眼旁观。流亡朝廷拒绝调兵，认为图拉尔只是“不成气候的蛮族”，且西部主要是山地贵族的私产，丢了也不心疼。这直接导致了后来西部防线的空虚。当图拉尔军队杀入西部腹地时，受害者全是山地贵族。流亡朝廷不仅不救，反而下令收缩防线保卫王都。其国家内部的政治斗争也愈演愈烈，腹背受敌的金诺瓦尔最终流干了鲜血，失去了作为战略屏障的能力。于是，一些守关的山地领主私下接触了王国，在大厦将倾之际，甚至主动打开了险峻的关隘，选择倒戈于他们的宿敌。在金诺瓦尔王都拉瑟陷落，其王室流亡泰梅里亚不知所踪后，斯佩比亚大公将拉瑟改名为“凯旋城”，标志着通向泰拉瑞安半岛的门户已然敞开一半。那些满口荣誉的平原流亡贵族追随王室逃往泰梅里亚，继续做他们的复国梦；而那些被视为粗鄙的山地新贵族，则摇身一变，成为了王国新设立的“金诺瓦尔总督区”下的市长、矿监和治安官。他们失去了国家，但保住了家族。
#### 费洛兰边陲国
**介绍**：费洛兰边陲国占据着佩雷迪尔山脉与冰海之间那条逼仄而致命的狭长地带。它是连接南方王国与泰拉瑞安诸国唯一的路上咽喉，这里便是天堑 **”七谷十二关“**。这里没有国王，只有由十二座关隘领主组成的政治团体所领导的**七谷议盟**。对于王国而言，北征若不经此地，便需绕行凶险的冰海或翻越不可逾越的佩雷蒂尔，代价数倍；但若强攻此地，则会陷入久战的泥淖，更将破坏南北的商路。

武装中立几乎成为了费洛兰人与生俱来的信条。他们向过往的商旅课税，向交战方兜售情报与雇佣兵。名义上，他曾是金诺瓦尔的附庸，但在金诺瓦尔遭到图拉尔与王国夹击而崩溃时，费洛兰不仅拒绝施以援手，反而趁火打劫，侵吞了金诺瓦尔东部的优良盆地。

维雷兰坐落在费洛兰走廊的偏北端，背靠冰海，扼守平原。它是“七谷十二关”中最宏伟、最坚固，也是资源最丰富的要塞都市。这里有着用黑花岗岩砌成的三重城墙，仓库里堆积着来自泰拉瑞安半岛的谷物与来自冰海的鲸油。在理论上，只要维雷兰关闭大门，它足以独自支撑数年的围城战，任何试图强攻此地的将领都会在它的城墙下流干鲜血。然而，这座号称“永不陷落”的冷石要塞，却在凯歌征服战争中成为了最大的讽刺。

当金诺瓦尔灭亡，王国大军越过前面的关卡，自金诺瓦尔的栈道压境之时，维雷兰的城守——一位传统的费洛兰老派骑士，誓言要让南方人付出惨痛代价。他动员了全城的兵力，甚至将滚烫的金汁与沥青搬上了城头。维雷兰的商业议会经过了精密的计算。但在开战的前夜，维雷兰的城市议会经过了一整夜精密的计算。他们算出了王国许诺的“贸易独家代理权”远比荣誉值钱。于是，当老城守在塔楼上擦拭他的传家宝剑，准备迎接第一缕晨曦中的敌军时，他身后的城门绞盘却发出了刺耳的转动声。 守军在惊愕中被自己的雇主缴械，王国军队在商会代表的红毯迎接下，兵不血刃地接管了这座北方锁钥。
```
“维雷兰人证明了一件事：只要价码合适，连城墙上的石头都会自己长腿跑掉。”
```

维雷兰的倒戈成为了多米诺骨牌的第一张。七谷十二关的天险瞬间成为了无用的摆设，剩余的领主不是因为恐惧战争，而是恐惧失去利益——他们唯恐王国将贸易特权独赐给维雷兰，于是争先恐后地向王国军队敞开了通往泰拉瑞安的大门。 最终，精明的领主们通过签署《银海和约》，彻底出卖了北方的盟友，以此换取了在新秩序下保留自治权与财富的特许。

```
“费洛兰的风有两种语言，半句是神圣的誓言，半句是甜美的交易。”
```
#### 夏尔赞侯国
**介绍**：波斯塔特同盟成员之一。伟哉夏尔赞。占据着泰拉瑞安半岛最肥沃的南部大平原与黄金海岸，夏尔赞侯国拥有同盟中最庞大的人口与财富。其都城晴夏尔赞曾经是整个大陆人口最多的超级都市。城中的每一块地砖都仿佛浸透了金粉。凭借着雄厚的财力，夏尔赞长期资助海盗袭扰王国海岸，是南方人最痛恨的“北境金库”。

同样痛恨他们的还有疍灵，不允许拥有任何干燥的地产。他们生老病死都必须在摇晃的船屋或发臭的淤泥棚屋里。备受欺压的种族因运送王国的军队，而获得了陆地上生存的权力，而不再仅仅只能在码头和野地中聚居。他们不仅运送军队，还拆掉了阻挡王国战船的水下暗桩。

同样痛恨他们的还有**疍灵**。这个被夏尔赞人蔑称为“半水鱼”的种族，长期遭受着“禁土令”——他们被禁止踏足城市的路面，生老病死只能蜷缩在潮湿的船屋与发臭的淤泥地里。 因此，当王国军队许诺给予他们“陆地上生存的权力”时，复仇的火焰点燃了整个水网。疍灵成为了王国最忠实的向导与运输队，他们熟悉每一条隐秘的水道，甚至亲自拆除了夏尔赞人布置在浅滩的防御暗桩，将敌军送入了夏尔赞南部城市的腹地。

富庶往往伴随软弱。当金诺瓦尔，维雷兰相继沦陷，盟友泰梅里亚倒戈相向，强大的夏尔赞沦落为孤岛。王国军队绕开了南部精密的防御体系，兵临夏尔赞都城之下。登时，那些不可一世的佣兵团在确定雇主已成孤岛后，他们不仅未发一矢便尽数作鸟兽散，临走前还洗劫了外城的金库。虽然夏尔赞大城拥有坚固的城防，但在没有任何希望的绝望围城战后，这座千年古城免不了陷落的命运。夏尔赞的沦陷，标志着凯歌征服战争的大体终结，也象征着波斯塔特法统的彻底断绝。

当王国军队攻入晴原夏尔赞时，并没有发生屠城。相反，斯佩比亚大公卸下盔甲，在全城与全军的注视下，与神圣古灵一同完成了象征古老帝国权柄授予的仪式。
#### 阿尔泽兰王国
**介绍**：波斯塔特同盟成员之一。这是位于半岛西北部走廊地带的佩雷蒂尔前山海岸王国。阿尔泽兰历史绵长，相比于阴谋与斗争不断地同盟，阿尔泽兰长期将国力投入到对抗东雪原上的国家与漫长海岸线上海盗的袭扰之中去。他们的审美清新淡雅，和重工华丽的波斯塔特诸国一直相去甚远。

当金诺瓦尔灭亡，费洛兰倒向王国之后，阿尔泽兰骤然发现通往其腹地的道路几乎已经畅通无阻。王国军队因为不熟悉水文，在“丰泽河谷”遭遇了春汛，损失惨重。后来王国军队吸取教训，利用丰泽地带的枯水期突袭，一举击溃阿尔泽兰同泰拉瑞安联盟的主力，古老的阿尔泽兰王国在随后轰然倒塌，其王室投降，因距离太远，仍然留存了下来。
## 波及势力

### 图拉尔山国
**介绍**：图拉尔山国是佩雷蒂尔山西麓的君主制国，信仰西方的琐罗亚教。虽然被称为“山国”，其实际领土位于山脉西侧的荒原，只是其扩张路径需翻越群山，故给东方诸国留下了“来自山中”的印象。

在MD15世纪早期，图拉尔在宗教领袖曼哥尔的带领下，利用魔晶技术改良了军队，并吸纳了大量东部流民，迅速向西扩张，建立了一个短暂而庞大的帝国。他们极具组织性，战场上伴随着高亢的宗教念唱，狂热的步兵方阵曾令金诺瓦尔人胆寒。 然而，其扩张最终因领袖在西征圣地途中遭雷击身亡而戛然而止，国家随后陷入分裂，退回沃勒利亚河（Volleria）以东。但他们留下的宗教火种，依然在佩雷迪尔山脉俩侧拥有燎原之势。
### 辉银龙国
**介绍**：自1353年从波斯塔特联盟中独立出来的半龙人与人类混居政权。由于拥有强大的龙团与银龙共主，且地理位置较为偏远， 虽然战火一度燃烧至其边境，但辉银龙国在战争初期保持了惊人的克制与中立，并且凭借精明的外交与强大的威慑力挫败了觊觎者。最终，在波斯塔特旧秩序彻底崩塌时，他们丝毫未损，甚至只是打开了大门，就接收了旧帝国一半的智慧遗产，大量学者和工匠的逃亡至此，带来了辉银随后的繁荣。

### 渎神教派支系
**介绍**： 这是一个在波斯塔特同盟衰落期兴起的宗教极端组织，被正统十神教会斥为“渎神教派”。他们实际上是西方琐罗亚教的激进分支，信奉**“烈火净世”。 他们宣称北境的连年战乱是“伪神”的惩罚，唯有纯净的烈火才能洗刷世间的罪恶。他们鼓动绝望的平民反抗领主与教会，导致了数次叛乱与冲击圣殿事件。

由于该教派成员也善用火焰神术，且常模仿“礼魂修会”身着遮面长袍（尽管多为杂色），当地百姓常混淆二者，称其为“戴帽卡琳”。这不仅是对礼魂修会的冒犯，也加剧了半岛宗教局势的混乱，直接导致了卡琳信徒对于其的重点打击。
### 礼魂修会
**介绍**： “战争非恶，死亡非终。” 他们不隶属于任何国家，而是战争之神卡琳的代行者之一。在这场绵延数十年的凯歌征服战争中，那一抹**白袍遮面**的身影贯穿始终。他们不参与杀戮，却比任何军队都更接近战场的中心。

 不同于主流教会对战争胜负的关注，礼魂僧侣专注于探讨战争与生命的终极奥义。他们身着白色长袍，头戴罩帽，口鼻掩于浸透香料的细布之下，穿梭在尸横遍野的荒弃战场。 他们是最高效的送葬者。利用娴熟的火焰神术与特制的香料，他们焚烧尸骸、阻断瘟疫、净化恶灵。在他们的指引下，破碎的灵魂得以安息，或为了完成未尽的遗愿而与其缔结契约，成为守护僧侣的“契灵”。

 礼魂修会不仅关注死者，也抚慰生者。在战火稍歇的间隙，他们会驻留在废墟般的村镇，救治伤患，重建家园。对于泰拉利安的百姓而言，这些神秘的白袍僧侣既是死亡的象征，也是这乱世中唯一的慈悲。
### 疍灵
**介绍**：疍灵是生活在冰海与半岛水网中的一类智慧生物。他们拥有适应水生的大眼睛，角膜有淡蓝反光膜，肢体带蹼，并且能长时间在水下闭气，他们的听觉极度敏锐，能辨识风声与水下共鸣。他们以船坞为家，随洋流迁徙，族群小而分散，每个船坞仅有几十上百人。每年夏季的“冰消盛会”是其族群最庞大的聚集时刻，那时，疍灵驾驶屋船前往集会，组成的海上船队规模甚至有小城之巨。他们的天敌是冰海东部的蛮子鱼人，蛮子鱼人抢掠为生，尤其中意于他们的船屋，疍灵深以为害。

长期以来，疍灵深受陆地种族的歧视与排挤，尤其是在夏尔赞，他们被视为“半水鱼”，严禁踏足城市路面。然而，在这场战争中，王国敏锐地意识到了他们的价值。通过许诺给予其“陆地生活权”，疍灵倒向了王国一方。 他们成为了王国海军在浅滩与内河最致命的向导。是疍灵潜入水下，徒手拆除了夏尔赞人引以为傲的拦海暗桩；也是疍灵驾驶着吃水极浅的快船，将斯佩比亚的步兵送入了水网密布的防线后方。
### 黑色巨龙
**介绍**：波斯塔特帝国的守护图腾。当旧帝国崩塌时，那条古老的守护巨龙随之殉国，其悲鸣曾响彻整个旷野（死的地方应该是王领东部）。 然而，龙的血脉并未断绝。她的子嗣被秘密带往泰拉瑞安的王廷孵化。随着诸国的覆灭，她族群的龙裔们部分追随他来到了此处，而仍然有部分龙裔，居留在先祖之地，也就是先进斯佩比亚领地的花林泽。
### 历史名词

###### 春秋袭劫
王国的常备军此次在春季进行作战骚扰，使得伯爵国困扰不堪。
王国的军队特别会在入春和秋日时发动军队进攻。因王国采用的常备军体系，在冬日的准备之后无需经过征兵便可在春天来临时发动进攻，而秋日对于泰拉米亚这一农业国家的打击更是不需准备太多辎重。这一长达十年的行为利用了王国东方领地的狮鹫与疍灵的快船。
###### 拉瑟和约
金诺瓦尔、费洛兰边陲国与斯佩比亚大公国签订的和平条约。
金诺瓦尔深陷图拉尔袭扰与内部斗争之虞，王国内部发生严重的权力斗争，费洛兰十二关则极力谋取利益，从中撮合。在斯佩比亚使者前往金诺瓦尔王城拉瑟后，三方顺利地签订了和约，为三方带来了接近四十余年的和平。
###### 银海和约
维雷兰要塞陷落后，本就软弱的七谷议会迅速向斯佩比亚投降，开放阻挡王国数百年的坚城，使得王国的铁骑穿行走廊畅行无阻，以换取日后各个城邦在半岛与王国贸易中的优惠与独占优势。

在王国东部领地的商业兴起后，王国西部日渐衰落，以至于王国商业绕过了“七谷十二关”。费洛兰不仅没得到承诺的贸易垄断，反而因为失去了战略价值，变成了一座被遗忘的“衰落城”，守着满仓库卖不出去的货物。那些精明的议员最终发现，和平条约上的墨水，比血迹干得更快。
###### 凯德利亚-金丝垒
凯德利亚（Caerdulia）是米瑞利亚时代以玉米面和蜂蜜或者糖调和后炸制而成的军粮。
选用优质玉米磨成粗面，掺入约两成的豆粉与少量小麦粉，再配合少量蛋液与盐水调和而成，然后将面团切条，投入滚沸的油脂中炸至金黄脱水。将炸酥的玉米条破碎成粗条后，趁热倒入熬至挂旗的糖浆中混合搅拌。混合物被填入标准化的矩形模具中高压压实，切块风干。最后送入烘炉进行二次低温烘焙，彻底逼出残留水分。用涂蜡的粗布包裹，可防潮保存数年之久，可以成为长距离奔袭使用的军粮。

战争结束后，这种配方流传入民间。商人们改良了配方，增加了昂贵的蜂蜜与糖比例，减少了硬度，并撒上坚果碎。这种改良版的“凯杜利亚”被人们称为“金丝垒”迅速成为了风靡的茶点。曾经让被征服者闻风丧胆的王国军队口粮，如今却成了平民也偶尔可以消费的美味点心。
